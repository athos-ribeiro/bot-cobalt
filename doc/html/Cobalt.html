NAME

Cobalt -- a pluggable IRC bot framework

DESCRIPTION

Cobalt is an IRC bot. Sort of.

The Cobalt core (this module) simply ties an event syndicator (POE::Component::Syndicator) into a configuration management system (using YAML), a logging system (Log::Handler), and some other useful wheels, so you don't have to reinvent them.

By default, the core will load the IRC.pm plugin to provide a single-server IRC interface, processing and re-broadcasting events from POE::Component::IRC.

Most actual work is done by plugins that exist in the plugin pipeline.

Cobalt is packaged with various plugins, most of which aim to replicate functionality found in darkbot and the original implementation of the cobalt IRC bot.

Plugins are simple to write but extremely flexible.

The core also provides some useful attributes, exposing configuration settings and state information to plugins in the pipeline. See "ATTRIBUTES" for more information.

There are some convenience methods available for accessing pieces of state information, managing timers, and so forth. See "METHODS" for information on core methods.
